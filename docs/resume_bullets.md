# Bullet points для резюме

## Версия для DS / ML

- Спроектировал end-to-end pet-project `IT Skills Radar`: ingestion вакансий, нормализованное хранилище в PostgreSQL, аналитические витрины, FastAPI API и Streamlit dashboard.
- Реализовал rule-based pipeline нормализации ролей и навыков для data/product вакансий без тяжелого NLP, включая обработку грязных title, skill aliases и неполных зарплатных данных.
- Построил SQL-витрины для анализа спроса на навыки, динамики навыков по времени и salary premium по навыкам на основе PostgreSQL views.
- Разработал API и UI для интерактивного анализа рынка junior / intern data-ролей и сравнения ролей по ключевым навыкам.
- Настроил воспроизводимую локальную среду через Docker Compose, автотесты на `pytest` и CI workflow в GitHub Actions.

## Версия для Product / Data Analyst

- Разработал портфельный аналитический сервис `IT Skills Radar` для исследования спроса на навыки в data/product вакансиях: от сбора данных до dashboard.
- Спроектировал понятную модель данных в PostgreSQL для вакансий, ролей, навыков, зарплат и временных признаков с учетом будущих аналитических запросов.
- Реализовал аналитические метрики: топ навыков по роли, помесячная динамика спроса, распределение навыков по ролям и salary premium по навыкам.
- Собрал FastAPI + Streamlit интерфейс для демонстрации аналитики с фильтрами по роли и уровню seniority.
- Добавил тесты, Docker-конфигурацию и CI, чтобы проект был воспроизводимым и пригодным для GitHub-портфолио.
